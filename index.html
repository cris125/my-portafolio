User Authentication(userDetailView.py)

This is the code of a /views/userSerializer.py project file which allows me to authenticate users with a token obtained by using the url login/ from the urls.py file "path('login/', TokenObtainPairView.as_view())" which gives the token for authentication when the user and password are entered.

if you want to see the complete code in the repository click here

from django.conf import settings
from rest_framework import generics, status
from rest_framework.response import Response
from rest_framework_simplejwt.backends import TokenBackend
from rest_framework.permissions import IsAuthenticated
from agenciaApp.models.user import User
from agenciaApp.serializers.userSerializer import UserSerializer
class UserDetailView(generics.RetrieveAPIView):
	queryset = User.objects.all()
	serializer_class = UserSerializer
	permission_classes = (IsAuthenticated,)
	def get(self, request, *args, **kwargs):
		token = request.META.get('HTTP_AUTHORIZATION')[7:]
		tokenBackend = TokenBackend(algorithm=settings.SIMPLE_JWT['ALGORITHM'])
		valid_data = tokenBackend.decode(token,verify=False)
		if valid_data['user_id'] != kwargs['pk']:
			stringResponse = {'detail':'Unauthorized Request'}
			return Response(stringResponse, status=status.HTTP_401_UNAUTHORIZED)
		return super().get(request, *args, **kwargs)

Below is the code utilized by the React component to initiate a login request to the Django back-end.

						
import axios from 'axios';
const API=  import.meta.env.VITE_API;
export const userAut = async (username, password) => {
    
    const response = await axios.post(`${API}/login/`, {
          username,
          password,
        });     
    localStorage.setItem('token_access', response.data.access);
    localStorage.setItem('token_refresh', response.data.refresh);    
    location.replace('/');
};

COMPONENT: INICIO

Within this React component, requests to the Django back-end are orchestrated using the access token obtained during the login process. Safely stored in local storage, this token acts as a secure key for accessing and modifying product information. The component effortlessly manages a spectrum of operations, encompassing the addition, modification, and deletion of products, offering an intuitive interface for comprehensive product management.
